Module Infection

module Infection: sig .. end
val random_check : int -> bool

random_check threshold picks a random int out of 100 and checks if it is less than the threshold. It returns a bool.

val start_tile_infection : Objects.Tile.t -> Objects.Tile.t

start_tile_infection tile starts an infection in tile tile (1 person infected) if it is not infected and does nothing to the tile if it is. It then returns the tile.

val infect_tile : Objects.Tile.t -> Objects.Disease.t -> Objects.Tile.t

infect_tile tile disease spreads the infection in tile tile if it contains any infected people (increases infected number). If tile is not infected, it does nothing.

val spread_to_neighbors : Objects.Tile.t array array ->
       int -> int -> int -> int -> Objects.Disease.t -> unit

spread_to_neighbors tiles row column rows cols disease spreads the disease from given tile at row row and column column to neighboring tiles based on a random spread probability given by disease disease.

val check_neighbors : Objects.Tile.t array array -> int -> int -> Objects.Disease.t -> unit

check_neighbors tiles row column disease checks if a tile at a given row and column in the map tiles is over the disease disease's infection threshold for spreading to neighboring tiles. If it is, the function randomly spreads to neighboring tiles based on the disease's random spread probability.
