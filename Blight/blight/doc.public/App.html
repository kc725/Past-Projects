Module App

module App: sig .. end
val string_to_list : string -> string list

string_to_list str splits a string str at every space into a list.

val read_command : State.State.t -> State.State.t

read_command st reads and parses in any commands whenever they are entered when the game is running.

val run_game : State.State.t -> 'a

run_game st is the main game loop that steps the game state st, spreads disease within and between tiles, and prints out the map.

val start_game : State.State.t -> string -> unit

start_game state start_coordinates starts the game with a given state state (with the map, disease, and civilizations) and starts the disease at a given location start_coordinates inputted by the user.

val condition_check : string -> int

condition_check checks is given in the range from 0 to 100. If not, the user is prompoted to try again

val coordinate_check : string -> int list

coordinate_check checks if coordinates are valid from a pair of numbers bigger than 0. If not, the user is prompoted again

val setup_disease : State.State.t -> unit

setup_disease state guides the user through the setup process (choosing a name, picking coordinates to place the disease, etc.).

val setup_game : unit

setup_disease lets the user choose default parameters about their disease.

val main : unit -> unit

main () starts the game and prompts the user for the starting coordinates of the disease.
