module State: sig .. end

Tracks the current state of the world.

val total_dead : State.t -> int

total_dead state calculates the total number of dead people in the world given by the state state.

val total_infected : State.t -> int

total_infected calculates the total number of infected people in the world given by the state state.

val total_population : State.t -> int

total_population calculates the total population in the world given by the state state.
